
DESCRIPTION:
--------------------------------

asprint is a command-line utility that can be used to print compiled
AppleScript (.scpt) files with nice formatting in the Terminal.

asprint requires Mac OS 10.5 or later.

(c) 2009-2010 Ali Rantakari
http://hasseg.org/asprint



INSTALLATION:
--------------------------------

You can run the install.command shell script to install asprint and
its man page under /usr/local/. If you'd like to install somewhere
else (e.g. under your home directory), you can just move the files
there yourself.
